    입니다
  2. input type="date" 날짜(년월일) 지정하는 폼요소
    입니다
    입니다
    (min="2023-02-17" max="2023-12-31" 최소, 최대값 지정)
  3. input type="datetime-local" 날짜와 시간(년월일 오전/오후 시 분)을 지정하는 폼요소
    입니다
  4. input type="week" 날짜(년주) 지정하는 폼요소
  5. input type="month" 날짜(년월) 지정하는 폼요소
  6. input type="number" 숫자를 지정하는 폼요소
    min="n"과 max="n"로 최소, 최대값 지정 가능
    프로젝트는 개 입니다
    프로젝트 인원은 최소 명 부터
    프로젝트 인원은 최대 명 입니다
  7. input type="range" 범위내의 값을 지정하는 폼요소
    (min="n"과 max="n"로 최소, 최대값 지정 가능)
  8. input type="tel" 연락처(전화번호) 지정하는 폼요소

    모바일브라우저에서 전화 걸기, 가상키패드 숫자형식 지원 가능
  9. input type="email" 연락처(이메일) 지정하는 폼요소

    브라우저에서 @유무, @뒤의 완성을 체크 해주며, 모바일브라우저에서 가상키패드 영문형식이나 @ 등 지원 가능
  10. input type="url" 연락처(웹주소) 지정하는 폼요소

    브라우저에서 프로토콜 유무 체크 해주며, 모바일브라우저에서 가상키패드 영문형식이나 .com 등 지원 가능
  11. input type="search" 검색단어를 지정하는 폼요소
  12. input type="color" 색상값을 지정하는 폼요소
