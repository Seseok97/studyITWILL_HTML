background 배경

  1. background-color: 색이름;
  2. background-image: url("이미지경로");

background-image의 속성들

  • background-repeat : 배경이미지의 반복 지정
    • repeat: 반복(x,y축 전부)
    • repeat-x: 가로로(x축)만 반복
    • repeat-y: 세로로(y축)만 반복
    • no-repeat: 반복없음.
  • background-size : 배경이미지의 크기 지정
    • cover: 이미지 하나로 영역 전체 덮음
    • contain: 이미지 늘림
    • n n
  • background-position : 배경이미지의 위치 지정
    • left/center/right top/center/bottom
    • Xn Yn
  • background-attachment : 배경이미지의 스크롤 여부 지정
    • scroll: 같이 움직인다.
    • fix: 고정되어있다.
    • local
