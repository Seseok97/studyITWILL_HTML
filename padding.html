padding - 요소의 내부 여백

border 기준 내부를 말한다.

1. padding의 방향별 지정속성

  • padding-top: n ; 위로 n만큼 적용
  • padding-right: n ;우로 n만큼 적용
  • padding-bottom: n ;아래로 n만큼 적용
  • padding-left: n ;좌로 n만큼 적용

2. padding의 값 갯수별 지정법

  • padding:n1 ;사방이 n만큼 적용
  • padding:n1 n2 ;상하n1 / 좌우n2 만큼 적용
  • padding:n1 n2 n3 ;상n1 / 좌우n2 / 하n3 만큼 적용
  • padding:n1 n2 n3 n4 ; 상n1 / 우n2 / 하n3 / 좌n4 만큼 적용
